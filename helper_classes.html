Helper classes and Components

Bootstrap comes with a whole bunch of helper classes and components that provide handy ready-to use css.

Buttons

The button (btn) classes let you format buttons and links as buttons. The default, primary, success, info, warning, and danger classes provide a selection of colors for your buttons. Since these labels are meaningful and semi-standard, its a good idea to use them where appropriate, and over-ride the color choices to match your color scheme.

Floats and Clears

The "pull-left" and "pull-right" classes are a fast way to float things to the left or right. Notice they don't include any padding, so you'll probably want to add an id or class, and add some in your own stylesheet.

Clearfix

The bootstrap framework gives us an easy way to clear unwanted floats. Adding the "clearfix" class to an element will clear floats below that element. I've used it here to prevent subsection below each example from wrapping around the image in the subsection above.
